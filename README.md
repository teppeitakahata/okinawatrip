# 沖縄旅のしおり

沖縄旅行用の、日ごとスケジュール型しおりアプリ。ビルド不要の静的HTML/CSS/JSのみで動作し、GitHub PagesでそのままPWA公開できます。

## 使い方

1. 「📍 行きたい場所」タブで、行きたいお店・スポットを登録する。
   - 住所を入力して「住所を検索」を押すと緯度経度を自動取得（OpenStreetMap Nominatimを利用、APIキー不要）。
   - 駐車場が複数あるお店などは「追加の場所」に第2駐車場等を個別に登録できる。それぞれ専用のGoogleマップリンクが表示される。
2. 右上の⚙️「旅の設定」で、拠点（ホテル・レンタカー営業所など）の住所・旅行日数・開始日・1日の活動時間帯を設定する。
3. 「🗺️ 日程」タブで「✨ AIで日程を自動作成」を押すと、拠点からの移動時間（直線距離ベースの推定）を考慮して、日ごとに近い場所をまとめて自動でスケジュールを組む。
4. 生成された予定は ▲▼ で順番入れ替え、✕で外す、プルダウンで別日へ移動、といった手動調整が可能。

## 技術構成

- ビルド不要の静的サイト（HTML/CSS/バニラJS、ESモジュール）
- データはブラウザのlocalStorageに保存（サーバー無し）
- 住所→緯度経度変換: OpenStreetMap Nominatim（無料・APIキー不要）
- 地図遷移: Googleマップの検索/ルートURLスキームを直接利用（住所テキストを優先、日本の番地表記もGoogle側で正しく解釈される）
- オフライン対応のService Worker（sw.js）でPWA化

## 移動時間の推定について

AIが提案する移動時間は、2点間の直線距離から沖縄の道路事情を踏まえた速度想定で計算した**参考値**です。実際の道路状況・信号・交通量により前後します。実際のルートや所要時間は各予定の「ここまでのルート」リンクから必ずGoogleマップ等でご確認ください。

## 更新時の注意

`sw.js` はオフラインキャッシュのため一度読み込んだファイルを強めにキャッシュします。デプロイ後に更新が反映されない場合は `sw.js` 先頭の `CACHE` のバージョン文字列（例: `okinawa-trip-v1` → `v2`）を上げてください。

## ローカルでの確認

```
python3 -m http.server 8080
```
を実行し、`http://localhost:8080` を開く。
